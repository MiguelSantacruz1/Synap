¿Qué es el Polimorfismo?

👉 Haz clic para revelar la respuesta

Respuesta

Es la capacidad de un objeto de tomar muchas formas. En OOP, permite que objetos de diferentes clases sean tratados como objetos de una clase común (generalmente a través de herencia o interfaces).
